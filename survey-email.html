Survey APIs (Public)

Tài liệu API tích hợp khảo sát dành cho MeviFarm & Bên thứ 3. (Không yêu cầu Authentication)

Môi trường & Base URL

Tất cả các API endpoint bên dưới đều được nối tiếp vào Base URL này. Vui lòng sử dụng đúng domain theo môi trường tích hợp của bạn.

https://api.xbest.io/lms-survey-service/v1 Production

Xem chi tiết bài khảo sát

Lấy thông tin định danh người dùng và danh sách câu hỏi trong kỳ khảo sát thông qua email.

GET /survey-periods/{surveyPeriodId}/results

Request Parameters

Parameter In Type Description
surveyPeriodIdRequired path integer ID của kỳ khảo sát
emailRequired query string Email của người tham gia khảo sát

Response Properties (Key Fields)

Field Type Description
status string Trạng thái của bài khảo sát:
not_started: Người dùng chưa làm hoặc chưa nộp.
submitted: Người dùng đã hoàn thành nộp bài.
resultQuestions array Danh sách các câu hỏi thuộc kỳ khảo sát.
answers array Nằm trong mỗi resultQuestions. Mảng này chứa các `id` của câu trả lời đã được sinh sẵn, FE cần dùng các `id` này để map dữ liệu khi nộp bài.

Response Example

application/json 200 OK

Nộp bài khảo sát

Gửi kết quả khảo sát. Yêu cầu truyền email trong body để đảm bảo tính định danh và toàn vẹn dữ liệu.

POST /survey-periods/{surveyPeriodId}/results/submit

Path Parameters

Parameter Type Description
surveyPeriodIdRequired integer ID của kỳ khảo sát

Body Parameters (application/json)

Parameter Type Description
emailRequired string Email định danh của người nộp bài.
dataSubmitRequired array Mảng chứa danh sách các câu trả lời.
questionIdRequired integer ID của câu hỏi (Lấy từ mảng resultQuestions ở API GET).
answersRequired array Mảng chi tiết đáp án cho câu hỏi này.
└answerIds array Mảng chứa ID của answer (ví dụ: [1824]). Lấy từ API GET.
└content string Nội dung chữ người dùng điền (đối với câu hỏi dạng tự luận/essay).

Request Example

application/json (Body Payload)

Response Example

application/json 200 OK
